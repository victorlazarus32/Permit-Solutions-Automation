Upload PRR File

Drop a public records request Excel export here. After upload, click Process PRR on the dashboard to ingest the rows into the system.

Workflow

  1. Pick the city the file came from.
  2. Choose the .xlsx file you received from the records office.
  3. Click Upload. The file lands in inbox/<city>/ on the server.
  4. Go to the Dashboard and click Process PRR to ingest the rows.
  5. The original file is auto-archived to inbox/processed/<city>/ with a timestamp.
